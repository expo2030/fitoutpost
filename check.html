How to open the correct index.html

1
Open File Explorer (Windows key + E)
2
Navigate to the Fitoutpost folder — the one containing check.html, index.html, build.py, etc.
3
Double-click index.html to open it fresh in your browser
4
If an old version still appears, press Ctrl + Shift + R to force-reload without cache
5
You should see a build date in the page footer (e.g. "Built 05 May 2026"). If you see "Make sure news.js is in the same folder", you have opened an old copy — go back to step 1.
